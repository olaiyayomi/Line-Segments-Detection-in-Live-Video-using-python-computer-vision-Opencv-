# Edge Detection and Line Segments in Live Video

This Python script utilizes OpenCV to perform real-time edge detection and identify line segments in the video stream from your computer's webcam. The Canny edge detection algorithm highlights edges, and the Hough Line Transform detects line segments, which are then drawn on the original frame in white.

## Prerequisites
- Python
- OpenCV library
